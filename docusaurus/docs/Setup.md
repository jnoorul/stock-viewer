# Setup Instructions


### What you'll need

  - Node.js version 18.0 or above:
  - Git CLI
  - Visual Studio Code or IntelliJ


## Clone the Starter Repo and Run

```bash
  git clone https://github.com/jnoorul/stock-viewer-starter.git
  cd stock-viewer-starter
  npm i
  npm run dev
```
